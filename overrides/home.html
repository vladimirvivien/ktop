Features

:material-chart-line: Real-Time Monitoring

Continuously updated view of nodes, pods, and containers with CPU and memory metrics.

:material-arrow-decision: Drill-Down Navigation

Navigate from cluster overview to node to pod to container logs with Enter/ESC.

:material-text-box-search: Container Logs

Stream, filter, and search container logs in real-time with full-screen mode.

:material-flash: Zero Installation

Single binary that runs locally. No agents or server-side components needed.

:material-kubernetes: Works Anywhere

Connects through the Kubernetes API. Works anywhere kubectl works.

:material-gauge: Multiple Metrics Sources

Prometheus for enhanced metrics, Metrics Server, or graceful fallback to requests/limits.

Quick Install

=== "kubectl plugin" ```bash kubectl krew install ktop kubectl ktop ``` === "Homebrew" ```bash brew tap vladimirvivien/oss-tools brew install ktop ``` === "Go install" ```bash go install github.com/vladimirvivien/ktop@latest ``` === "Binary" Download from [GitHub Releases](https://github.com/vladimirvivien/ktop/releases/latest)
